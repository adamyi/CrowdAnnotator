                                In this task, you will read an instant messaging conversation and annotate words or groups of words that are potentially sensitive. We have provided a few examples on the left and a list of data categories for you to annotate. For example, some word/group of words may represent name of a person/organization/place, or date/time, or username/password and so on. For each of such word/group of words: 
                                

                                
1) Select the text portion, and annotate the desired category by clicking the appropriate label.
                                
2) If there is no appropriate label, you can add a new one by clicking Add New.
                                
3) If you want to undo an annotation, select the annotation click Undo.
                                
4) Click the Next Task button to submit annotations and proceed to the next task.
                                

                                
To see some examples, please refer to the left bar.
                                
For each annotation, please do not annotate more than you should. For example, do not annotate "email:" as part of the email annotation for a message "email: xxx@yyy.zzz". 
                                
NOTE: You cannot skip tasks, and will be only paid for the first 3 tasks (not including the warm-up task at the beginning) in this HIT.
